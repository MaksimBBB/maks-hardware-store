# JSON API - Магазин техніки

Цей проєкт — простий JSON API, який імітує роботу магазину техніки.
API дозволяє додавати товари, отримувати список товарів, отримувати товар за ID
та видаляти товар. Дані зберігаються у JSON-файлах.

## Запуск проєкту
1. Переконайтесь, що Go встановлений (версія 1.18+)
2. Перейдіть у папку з проєктом
3. Запустіть сервер командою: `go run main.go`

4. Сервер запуститься на порту 8080


## Тестування через Insomnia

GET /list
Повертає список усіх товарів у форматі JSON

POST /add
Додає новий товар
Очікує JSON у тілі запиту:
`
{
  "name": "Laptop",
  "brand": "HP",
  "price": 38000
}
`

GET /item/{id}
Повертає товар за його ID

DELETE /delete/{id}
Видаляє товар за ID


## Приклади відповідей
Успішне додавання:
`{
  "id": 3,
  "name": "Phone",
  "brand": "Samsung",
  "price": 17000
}`

Помилка:
`{"error":"invalid JSON"}`
